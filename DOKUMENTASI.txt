1. INFORMASI UMUM
--------------------------------------------------------------------------------
   Nama Aplikasi  : MONITORINGv2 - Sistem Monitoring PKL
   Institusi      : SMKN 1 Subang
   Versi          : 2.0
   Tahun          : 2026
   Platform       : Web Application (SPA)
   Tujuan         : Mendigitalisasi seluruh proses monitoring Praktik Kerja
                    Lapangan (PKL) siswa SMKN 1 Subang secara real-time.


2. FITUR UTAMA
--------------------------------------------------------------------------------

   A. Manajemen Pengguna
      - Multi-role: Admin, Siswa, Guru, dan Perusahaan (DUDI)
      - Registrasi dengan sistem approval
      - Manajemen profil lengkap (foto, password, dll.)

   B. Absensi Berbasis GPS (Geofencing)
      - Check-in dan check-out menggunakan lokasi real-time
      - Validasi radius menggunakan rumus Haversine
      - Batasan radius yang dapat dikonfigurasi
      - Foto saat check-in/check-out
      - Status: Hadir, Terlambat, Alpha, Sakit, Izin

   C. Jurnal Harian (Logbook)
      - Catatan kegiatan harian siswa selama PKL
      - Lampiran file (gambar, PDF, dokumen)
      - Review dan penilaian oleh Guru & Perusahaan
      - Status: Pending, Disetujui, Ditolak

   D. Perizinan & Sakit
      - Pengajuan izin/sakit dengan bukti lampiran
      - Approval oleh Guru pembimbing
      - Riwayat pengajuan lengkap

   E. Penempatan Siswa (Placement)
      - Penugasan siswa ke perusahaan tertentu
      - Penunjukan guru pembimbing
      - Periode PKL (tanggal mulai & selesai)

   F. Penilaian Multidimensi
      - Nilai Absensi
      - Nilai Logbook
      - Nilai Laporan
      - Sikap
      - Kinerja
      - Kedisiplinan
      - Penilaian oleh Guru & Perusahaan

   G. Peta Sebaran Interaktif
      - Visualisasi lokasi perusahaan di peta (Leaflet)
      - Sebaran siswa per perusahaan
      - Informasi detail setiap titik lokasi

   H. Laporan & Ekspor Excel
      - Rekap absensi
      - Rekap logbook
      - Ringkasan siswa
      - Nilai akhir
      - Format file: Excel (.xlsx)

   I. Notifikasi In-App
      - Pemberitahuan real-time
      - Approval/penolakan registrasi
      - Pengingat kegiatan

   J. Laporan Akhir PKL
      - Upload laporan akhir (file)
      - Review oleh Guru pembimbing
      - Riwayat revisi

6. CARA INSTALASI & MENJALANKAN
--------------------------------------------------------------------------------

   Prasyarat:
   - PHP >= 8.3
   - Composer
   - Node.js & npm
   - MySQL 8.x
   - Laragon / XAMPP / WAMP

   Langkah-langkah:

   1. Clone repositori
      git clone <repository-url>
      cd monitoring-pkl-backend

   2. Install dependensi backend
      composer install

   3. Konfigurasi environment
      cp .env.example .env
      - Sesuaikan konfigurasi database di file .env
      DB_DATABASE=monitoring_pkl
      DB_USERNAME=root
      DB_PASSWORD=

   4. Generate key aplikasi
      php artisan key:generate

   5. Buat database
      mysql -u root -e "CREATE DATABASE monitoring_pkl"

   6. Jalankan migrasi
      php artisan migrate

   7. (Opsional) Import data awal
      php artisan db:seed

   8. Install dependensi frontend
      cd ../monitoring-pkl-frontend
      npm install

   9. Jalankan aplikasi
      Terminal 1: php artisan serve (di folder backend)
      Terminal 2: npm run dev (di folder frontend)

   Akses:
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:8000/api

8. ALGORITMA GEOFENCING (Haversine Formula)
--------------------------------------------------------------------------------

   Rumus yang digunakan untuk memvalidasi jarak antara lokasi siswa
   dengan lokasi perusahaan saat absensi:

   a = sin²(��lat/2) + cos(lat1) * cos(lat2) * sin²(��long/2)
   c = 2 * atan2(��a, ��(1-a))
   d = R * c

   Keterangan:
   - R = Radius bumi (6371 km)
   - lat1, long1 = Koordinat siswa
   - lat2, long2 = Koordinat perusahaan
   - d = Jarak antara dua titik

   Jika d <= radius yang ditentukan -> Lokasi valid (dapat absensi)
